# 多页面静态网站生成器
> webpack、babel、sass、支持多页面、html/js/css分离、html美化、cdn地址替换、自动刷新、全局jQuery变量、打包历史备份

## 使用

安装

```sh
$ git clone https://github.com/zhw2590582/web-generator.git
$ cd web-generator
$ npm install
```

开发

```sh
$ npm run dev
```

添加

```sh
$ npm run add pageName
```

打包

```sh
$ npm run build
```

发布

```sh
$ npm run publish
```
